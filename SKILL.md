# 文档转结构化教程 Skill

将原始文档资料转化为「100岁老奶奶都能看懂、跟着操作」的模块式教程。

## 核心原则

1. **实操优先**：每个步骤都要具体到「点哪里、填什么」
2. **口语化表达**：像面对面教人一样，用「你」「咱们」「先...再...」
3. **模块化结构**：大主题拆成模块，模块拆成课时，层层递进
4. **主动增强**：搜索补充官方最新信息和典型案例

## 转化流程

### 第一步：分析原始素材

1. 通读全文，提取核心知识点
2. 识别内容类型：概念讲解 / 操作步骤 / 案例分析 / 注意事项
3. 标记过时或存疑的内容（后续搜索验证）

### 第二步：搜索补充

针对以下内容主动搜索补充：

1. **官方最新政策**：搜索「[平台名] 官方 [功能名] 最新政策 site:*.tiktok.com OR site:seller*.tiktok.com」
2. **操作界面变化**：搜索「[平台名] [功能] 2024/2025 最新教程」
3. **典型案例**：搜索「[领域] 成功案例 [平台名]」

补充内容融入正文，标注来源格式：
```
（来源：TikTok 官方帮助中心）
（来源：TikTok Shop 卖家中心）
```

### 第三步：重构结构

1. 按逻辑关系划分模块（入门→基础→进阶→高级）
2. 每个模块拆分 2-5 个课时
3. 每个课时聚焦一个完整任务

### 第四步：改写内容

**原文示例：**
> TikTok账号注册需要准备海外手机号或邮箱，建议使用Gmail邮箱注册，注册时需要填写生日信息，建议设置为成年人生日以获得完整功能。

**改写后：**
> 注册 TikTok 账号之前，你需要准备一个邮箱——推荐用 Gmail，稳定不容易出问题。
>
> 第一步，打开 TikTok App，点击底部的「我」。
>
> 第二步，选择「用邮箱注册」，输入你的 Gmail 地址。
>
> 第三步，设置生日——【注意】这里一定要填 18 岁以上的日期！填成未成年人的话，很多功能会被限制，比如直播、私信都用不了。
>
> 【踩坑】我们团队实测过，如果生日填错了，后面改不了，只能重新注册。所以第一次就填对！

### 第五步：添加增强元素

为每个课时添加：
- 【重点】标记核心知识点
- 【注意】标记容易忽略的细节
- 【踩坑】标记常见错误和正确做法
- ✅ Checklist（5-10 项，可勾选）

为每个模块添加：
- 🎯 学习目标（3-5 条）
- 📝 模块小结
- 🏋️ 课后作业（可选）

## 语言风格规范

### ✅ 推荐用法

| 场景 | 推荐表达 |
|------|----------|
| 引导操作 | 「第一步，打开...」「接下来，点击...」「然后你会看到...」 |
| 强调重点 | 「这里划重点」「一定要记住」「最关键的是」 |
| 解释原因 | 「为什么呢？因为...」「这样做的好处是...」 |
| 举例说明 | 「比如说...」「举个例子...」「就像...」 |
| 经验分享 | 「我们团队实测过...」「根据经验...」「实际操作中...」 |
| 过渡衔接 | 「搞定这个之后...」「做完上面这些...」「下一步...」 |

### ❌ 避免用法

| 避免 | 原因 |
|------|------|
| 「您」「贵」 | 太正式，用「你」更亲切 |
| 「进行 XXX 操作」 | 啰嗦，直接说「点击 XXX」 |
| 「相关内容」「有关事项」 | 模糊，要具体 |
| 长句套长句 | 难懂，拆成短句 |
| 被动语态 | 「被设置为」→「设置成」 |

## 标记系统

```markdown
【重点】核心知识点，必须记住的内容

【注意】容易忽略的细节，操作时要留意

【踩坑】常见错误 + 正确做法，避免走弯路

> 💡 小技巧：提升效率的方法（来源：XXX）
```

## 质量检查清单

转化完成后，检查以下项目：

- [ ] 每个操作步骤是否具体到「点哪里」
- [ ] 是否有模糊表达（「相关设置」「某些情况」）
- [ ] 专业术语是否有解释
- [ ] 是否搜索补充了官方最新信息
- [ ] 每个课时是否有 Checklist
- [ ] 每个模块是否有学习目标和小结
- [ ] 口语化程度是否足够（读出来顺不顺）
